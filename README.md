Файл correlation.py содержит расчет корреляций каждой пары акций и построение корреляционных матриц. 
В зависимости от того, какую выборку (малую или большую) используем, комментируется малый или большой список тикеров.

Файл graph.py строит пороговые графы с заданными порогами на основе построенных ранее корреляционных матриц, а также MST для обеих выборок.

Файл comparison.py содержит построение сравнительных таблиц. Таблицы попарно сравнивают пороговые графы разных коэффициентов с одним порогом (например, 0.1 Кендалл с 0.1 Спирмен, 0.5 Спирмен с 0.5 Пирсон и т.д.). 
Представленный код сравнивает графы большой выборки. Для использования на малой выборке необходимо во всех путях заменить "50" на "10".
